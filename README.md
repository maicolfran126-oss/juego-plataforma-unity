# Juego Plataforma en Unity 🎮

## Descripción
Videojuego 2D tipo plataforma donde el jugador debe evitar enemigos y superar obstáculos.

## Características
- Movimiento y salto
- Enemigos
- Sistema de colisiones
- Interfaz básica

## Tecnologías
- Unity
- C#
